| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, badge inner padding |
| sm | 8px | Compact element spacing, section gap |

| lg | 24px | Section padding, header padding (p-6) |

Source: tailwind.config.js default spacing scale (4px base), existing component patterns.

## Glass Morphism Contract (Defining Visual Signature)

Every user-facing modal and elevated surface MUST use this glass pattern. Do NOT use opaque solid backgrounds.

### Modal/Card Glass Pattern
```
bg-white/90 dark:bg-gray-900/90
backdrop-blur-xl
border border-white/50 dark:border-white/[0.08]
shadow-[0_8px_40px_rgb(0,0,0,0.12)] dark:shadow-[0_8px_40px_rgb(0,0,0,0.4)]
ring-1 ring-black/5 dark:ring-white/10
rounded-3xl
```

### Input Glass Pattern
```
bg-white/50 dark:bg-white/[0.03]         (standalone inputs)
bg-white/70 dark:bg-gray-900/70          (input rows in bar context)
border border-gray-200 dark:border-white/[0.08]
rounded-xl
focus:ring-2 focus:ring-blue-500/30 focus:border-blue-400
```

### Fixed Border Rules (DO NOT DEVIATE):
- Light mode borders: `border-gray-200`, `border-gray-200/60`, or `border-gray-200/70`
- Dark mode borders: ALWAYS `dark:border-white/[0.08]` (never `dark:border-gray-700`)
- Dark mode hover borders: `dark:border-white/[0.18]`
- Separator: `bg-gray-200 dark:bg-white/[0.08]` (1px height)

### Radius Conventions:
| Element | Class | Value |
|---------|-------|-------|
| Small badges/tags | `rounded` | 4px |
| Buttons, inputs, selects | `rounded-xl` | 12px |
| Cards, dropdown panels | `rounded-xl` | 12px |
| Modal panels | `rounded-3xl` | 24px |
| Admin dashboard cards | `rounded-2xl` | 16px |

Source: ui-design-system memory -- this is THE defining visual signature.

## Animation Contract

All entrance animations use the canonical easing curve `cubic-bezier(0.16, 1, 0.3, 1)`.

| Class | Effect | Duration | Where Used |
|-------|--------|----------|------------|
| `animate-overlay-in` | Fade in | 0.2s ease-out | Modal backdrops |
| `animate-modal-in` | scale(0.95)->1 + translateY(10px)->0 + fade | 0.25s | All modal panels |
| `animate-dropdown-down` | scaleY(0.9)->1 + translateY(-4px)->0 | 0.15s | Dropdown menus |

All modals use this overlay + panel pattern:

```
  <div className="relative z-10 w-full max-w-md rounded-3xl ... animate-modal-in" onClick={e => e.stopPropagation()}>

```

Source: src/index.css keyframes, existing modal implementations.
